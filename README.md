
# SaaS CRM Dashboard (React + Supabase)

A modern SaaS CRM dashboard with role-based authentication, lead management, and admin control system.

---

##  Live Demo
https://saa-s-crm-dashboard-react.vercel.app

---

##  Features

###  Authentication
- Supabase Auth integration
- Protected routes
- Role-based access (Super Admin / Admin / User)
- Demo login (one-click super admin login)

###  CRM System
- Add, edit, delete leads
- Lead status workflow (New → Contacted → Converted)
- Search & filter system

### 👨‍💼 Admin Panel
- User management system
- Role switching (admin/user)
- Super admin protection

### UI/UX
- Modern SaaS dashboard design
- Fully responsive layout
- Dark mode support
- Clean modal-based UI system

---

## 🛠 Tech Stack
- React
- TypeScript
- Tailwind CSS
- Supabase (Auth + Database)
- Vercel (Deployment)
